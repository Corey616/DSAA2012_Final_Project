# ADR-001: SCA Window Attention
- **日期**: 2026-04-27
- **状态**: accepted
- **背景**: 全帧 SCA (所有帧互相attend) 导致帧差异过小，
  画面过于一致，缺乏多样性。Baseline 帧差异仅 43/37。
- **决策**: 改用滑动窗口注意力 (±1 neighbor) + 分层强度 (down=0.7, mid=0.5, up=0.2)。
  窗口外帧不参与 K/V 扩展，保留多样性。
- **后果**: 帧多样性 +37% (43→64), Consistency 从 0.565 降至 0.413 目标区间。
